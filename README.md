# 📊 Análisis de Ventas con Power BI – Droguería C.I.

## Descripción del proyecto

Este proyecto consiste en el desarrollo de un dashboard de Business Intelligence en Power BI para analizar el desempeño comercial de las ventas de una droguería durante el cuarto trimestre del 2024 (octubre – diciembre).

El análisis busca explorar el comportamiento de las ventas, identificar los productos más vendidos, analizar el rendimiento de los vendedores y evaluar el impacto de las devoluciones en los resultados comerciales.

Este proyecto fue desarrollado como parte de mi formación en Power BI y análisis de datos.

---

## Objetivos del análisis

- Analizar el comportamiento de las ventas durante el trimestre.
- Identificar los productos con mayor participación en las ventas.
- Evaluar el desempeño de los vendedores.
- Analizar los principales clientes de la empresa.
- Identificar el impacto de las devoluciones en las ventas.

Dentro del dashboard se analizaron los siguientes indicadores:

- Ventas brutas
- Ventas netas

- Ventas por cliente
- Ventas por localidad

---

## Principales hallazgos

Durante el análisis se identificaron algunos puntos importantes:

- Las ventas presentan una disminución en los dos últimos meses del trimestre** en comparación con octubre.
- Las notas de crédito por devoluciones representan aproximadamente el 7.41% de las ventas netas.
- Existe una alta concentración de ventas en un vendedor**, quien genera más del 50% de las ventas totales.
- EsSalud es el principal cliente, con el mayor volumen de unidades compradas.
- El producto con mayor participación en las ventas fue Ganciclovir como sal sódica de 500 mg.
